Job Title: Senior Software Engineer (Backend / Distributed Systems)

📍 Austin, Texas (Hybrid)
🏢 Industry: FinTech / SaaS
🧑‍💻 Experience Required: 7–10 Years

Role Overview

We are looking for a Senior Software Engineer to design, develop, and scale highly available backend systems that power mission-critical financial and transactional platforms. The ideal candidate has strong experience in distributed systems, microservices architecture, and cloud-native development.

You will work on systems that handle millions of daily transactions, requiring high reliability, performance, and security. This role involves close collaboration with product, QA, and infrastructure teams to deliver scalable and maintainable solutions.

Key Responsibilities
System Design & Architecture
Design and implement scalable backend services using microservices architecture
Architect systems for high availability, fault tolerance, and low latency
Evaluate trade-offs between monolithic and distributed architectures
Drive design discussions and contribute to long-term technical strategy
Backend Development
Build and maintain RESTful and/or GraphQL APIs
Develop services using languages such as Python, Java, or Go
Optimize database queries and data access patterns for performance
Implement caching strategies (Redis or similar) to improve response times
Cloud & Infrastructure
Design and deploy applications on AWS (EC2, S3, Lambda, RDS, ECS)
Containerize services using Docker and orchestrate using Kubernetes
Use Infrastructure as Code tools like Terraform for environment setup
Monitor system performance and reliability using observability tools
Performance & Reliability
Analyze system bottlenecks and optimize for latency and throughput
Implement retry logic, circuit breakers, and failover strategies
Ensure systems meet uptime targets (99.9%+)
Conduct load and stress testing using tools like JMeter
DevOps & CI/CD
Build and maintain CI/CD pipelines using Jenkins or GitHub Actions
Automate testing, deployment, and rollback strategies
Improve deployment frequency and reduce failure rates
Collaborate with QA teams to enhance test coverage and automation
Security & Compliance
Implement secure coding practices and follow industry standards (e.g., PCI-DSS)
Participate in security reviews and vulnerability assessments
Ensure data protection and compliance across services
Collaboration & Leadership
Mentor junior engineers and conduct code reviews
Work cross-functionally with product managers, QA, and DevOps teams
Participate in sprint planning, retrospectives, and design reviews
Drive engineering best practices across the team
Required Qualifications
7+ years of experience in backend software development
Strong proficiency in at least one language: Python, Java, or Go
Experience with microservices and distributed systems design
Hands-on experience with AWS or similar cloud platforms
Solid understanding of databases (SQL and NoSQL)
Experience with Docker and Kubernetes
Familiarity with CI/CD pipelines and DevOps practices
Strong problem-solving and debugging skills
Preferred Qualifications
Experience in fintech or payment systems
Knowledge of event-driven architecture (Kafka or similar)
Familiarity with observability tools (Prometheus, Grafana, OpenTelemetry)
Experience with high-throughput, low-latency systems
Understanding of security standards and compliance frameworks
Nice-to-Have
Prior experience working on systems handling large-scale transactions
Contributions to open-source projects
Experience with machine learning integration in backend systems
Public speaking, technical blogging, or knowledge-sharing experience

6–12 Months:
